HTML 활용

구글링을 열심히 하면서 HTML과 CSS에 익숙해지고 있음
CSS를 통해 HTML 배경화면 색 바꾸고 폰트 바꿀 수도 있음

오늘은 더 심화해서 회원가입 창을 만들어 보려고 함
일단 처음보는 태그인 [form] 태그를 발견함
데이터를 제출하기 위한 웹 양식 영역이라고 함
주요 태그로는 input, textarea, select, button 등이 있음
속성에는 action. method, novalidate, target, enctype이 있음

action - 데이터를 전송할 url 지정함
따로 url을 지정하지 않으면 현재의 url로 전송됨

method - 데이터 제출 방법을 지정함
종류로는 get, post, dialog가 있음
get은 기본값으로 쿼리 문자열로 전송,
post는 http요청 body에 포함하여 전송
dialog는 양식 영역이 dialog 내에 있는 경우 대화상자를 닫고
데이터를 제출하거나, 양식을 지우지 않고 제출 시 제출 이벤트가 실행되도록 함
주로 상자 안에 있는 웹 양식 영역의 데이터를
상자의 반환값으로 설정할 때 시용한다고 함

novalidate - 바로 예제를 보자!
